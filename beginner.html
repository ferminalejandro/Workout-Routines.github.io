Beginner Workout Plan

Monday

Leg Press: 3 x 6-8 Reps

Chest-Supported T-Bar Row: 3 x 8-10 Reps

Glute Ham Raise: 2 x 8-10 Reps

Dumbbell Lateral Raise: 3 x 12-15 Reps

Pec Deck: 2 x 15-20 Reps

Preacher Curl: 2 x 12-15 Reps

Wednesday

Machine Chest Press: 3 x 6-8 Reps

Leg Press: 3 x 6-8 Reps

Lat Pulldown: 3 x 8-10 Reps

EZ Bar Skullcrusher: 2 x 12-15 Reps

Reverse Pec Deck: 2 x 15-20 Reps

Seated Calf Raise: 3 x 10-12 Reps

Friday

Romanian Deadlift: 3 x 8-10 Reps

Incline Dumbbell Press: 3 x 8-10 Reps

Seated Leg Curl: 2 x 10-12 Reps

Lat Pulldown: 3 x 10-12 Reps

Leg Extension: 2 x 10-12 Reps
